Problem 1 – Move Zeroes to End
Problem Statement

We need to move all the zeroes present in the array to the end while maintaining the relative order of the non-zero elements.

Approach

To do this we will follow these steps:

We will find the index of the first zero in the array.
Then we will traverse the remaining elements.
Whenever we find a non-zero element, we will swap it with the zero element.
After every swap, we will move the zero pointer to the next position.
Code
#include<iostream>
using namespace std;

int main()
{
    int array[]={1,0,2,3,0,4,0,1};
    int n=8;

    int j=-1;

    for(int i=0;i<n;i++)
    {
        if(array[i]==0)
        {
            j=i;
            break;
        }
    }

    if(j==-1)
    {
        for(int i=0;i<n;i++)
            cout<<array[i]<<" ";
        return 0;
    }

    for(int i=j+1;i<n;i++)
    {
        if(array[i]!=0)
        {
            swap(array[i],array[j]);
            j++;
        }
    }

    for(int i=0;i<n;i++)
        cout<<array[i]<<" ";
}
Example

If we have the array

1 0 2 3 0 4 0 1

After moving all the zeroes to the end, the array becomes

1 2 3 4 1 0 0 0

This is because all the non-zero elements keep their original order while every zero is moved to the end.

Time Complexity

The time complexity of this solution is

O(n)

This means that every element is visited only once, so the running time increases linearly with the size of the array.

Space Complexity

The space complexity of this solution is

O(1)

This means that no extra array is used, so the space required remains constant.

Problem 2 – Linear Search
Problem Statement

We need to find whether a given element is present in an array.

Approach

To do this we will follow these steps:

We will start traversing the array from the first element.
We will compare each element with the target element.
If the target element is found, we will return its index.
If we reach the end of the array without finding the element, we will return -1.
Code
#include<iostream>
using namespace std;

int LinearSearch(int array[], int n, int target)
{
    for(int i=0;i<n;i++)
    {
        if(array[i]==target)
            return i;
    }

    return -1;
}

int main()
{
    int array[]={2,4,6,8,10};
    int n=5;
    int target=8;

    cout<<LinearSearch(array,n,target);
}
Example

If we have the array

2 4 6 8 10

and the target element is

8

The output will be

3

This is because 8 is present at index 3 in the array.

Time Complexity

The time complexity of this solution is

O(n)

This means that in the worst case we may have to check every element of the array before finding the target element.

Space Complexity

The space complexity of this solution is

O(1)
