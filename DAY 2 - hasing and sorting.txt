# Day 02/100 – Hashing & Sorting

## 📌 What is Hashing?

Hashing is a technique used to store and retrieve data efficiently.

Instead of searching every element one by one (Linear Search), we store the frequency of elements using an array or a hash map.

Hashing helps answer queries in **O(1)** average time.

Example:

Array

1 2 1 3 2 1 5

Frequency Table

1 → 3

2 → 2

3 → 1

5 → 1

---

## Types of Hashing

### 1. Array Hashing

Used when numbers are within a limited range.

Example

```cpp
int hash[100] = {0};

Time Complexity

Building Hash : O(n)

Searching : O(1)

Space Complexity

O(maximum value)
2. Character Hashing

Used to count lowercase alphabets.

There are only 26 lowercase letters (a-z).

Formula

hash[s[i]-'a']++;

Example

apple

a ->1

p ->2

l ->1

e ->1

#include<iostream>

        cin>>arr[i];

    int hash[100]={0};

        hash[arr[i]]++;

    {
        int number;

Problem 3 – Count Frequency using unordered_map
Code
#include<iostream>
#include<unordered_map>

using namespace std;

int main()
{
    int n;
    cin>>n;

    unordered_map<int,int> mp;

    for(int i=0;i<n;i++)
    {
        int x;
        cin>>x;
        mp[x]++;
    }

    for(auto it:mp)
    {
        cout<<it.first<<" "<<it.second<<endl;
    }
}

Selection step
Idea

Find the minimum element and swap it with the current index.

Code
#include<iostream>
using namespace std;

int main()
{
    int arr[]={13,46,24,52,20,9};

    int n=6;

    for(int i=0;i<n-1;i++)
    {
        int mini=i;

        for(int j=i+1;j<n;j++)
        {
            if(arr[j]<arr[mini])
                mini=j;
        }

        swap(arr[i],arr[mini]);
    }

    for(int i=0;i<n;i++)
        cout<<arr[i]<<" ";
}

Sorting step
Sorting step
Idea

Compare adjacent elements and swap if they are in the wrong order.

Largest element moves to the end after every pass.

Code
#include<iostream>
using namespace std;

int main()
{
    int arr[]={13,46,24,52,20,9};

    int n=6;

    for(int i=n-1;i>=0;i--)
    {
        bool didSwap=false;

        for(int j=0;j<i;j++)
        {
            if(arr[j]>arr[j+1])
            {
                swap(arr[j],arr[j+1]);
                didSwap=true;
            }
        }

        if(didSwap==false)
            break;
    }

    for(int i=0;i<n;i++)
        cout<<arr[i]<<" ";
}

Insertion step
Idea

Take one element and insert it into its correct position in the sorted part of the array.

Code
#include<iostream>
using namespace std;

int main()
{
    int arr[]={13,46,24,52,20,9};

    int n=6;

    for(int i=0;i<n;i++)
    {
        int j=i;

        while(j>0 && arr[j-1]>arr[j])
        {
            swap(arr[j],arr[j-1]);
            j--;
        }
    }

    for(int i=0;i<n;i++)
        cout<<arr[i]<<" ";
}

9 13 20 24 46 52
