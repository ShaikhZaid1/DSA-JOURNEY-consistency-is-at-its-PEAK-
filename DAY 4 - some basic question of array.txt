#  Problem 1 – Largest Element in an Array

#problem Statement

We need to find the element that is present in an array.

# Approach

To do this we will follow these steps:

- We will assume that the first element is the element in the array.

- Then we will go through the array.

- If we find an element that's bigger than the largest element we have found so far we will update the largest element.

#Code

cpp

#include<iostream>

using namespace std;

int main()

{

int array[]={2,5,1,3,0};

int largest=array[0];

for(int i=1;i<5;i++)

{

if(array[i]>largest)

largest=array[i];

}

cout<<largest;

}

# Example

If we have the arra

2 5 1 3 0


The largest element in the array is

5


This is because 5 is the number in the array.

# Time Complexity

The time complexity of this solution is

O(n


This means that the time it takes to find the largest element will increase linearly with the size of the array.

# Space Complexity

The space complexity of this solution is

O(1)

This means that the space required does not change with the size of the array so it is constant

#      Problem 3 – Check if Array is Sorted

# Problem Statement

We need to check whether the array is sorted in -decreasing order.

#### Approach

To do this we will follow these steps:

- We will compare every element with the element.

- If we find any element that's greater than the next element we will say that the array is not sorted.

## Code
cpp

#include<iostream>

{

int array[]={1,2,3,4,5};

#        Problem 4 – Remove Duplicates from Sorted Array

# Problem Statement

We need to remove elements from a sorted array.

# Approach

To do this we will follow these steps:

- We will maintain two pointers:

I → last unique element

J → element

- We will copy only unique elements to the front of the array.

#### Code

cpp

#include<iostream>

using namespace std;

int main()

{

int array[]={1,1,2,2,2,3,3};

int i=0;

for(int j=1;j<n;j++)

if(array[i]!=array[j])

{

i++;

array[i]=array[j];

}

}

for(int k=0;k<=i;k++)

cout<<array[k]<<" ";

}
